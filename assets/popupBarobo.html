LinkSense

Barobo is a children's robotics company that sells a modular robot system called Linkbot. However, Barobo does not sell sensors to go with the robot kits. I designed the LinkSense for Barobo, which is an adapter that allows the Linkbot to connect to and use Lego Mindstorms NXT sensors. The finished device uses an Arduino Pro Mini and has a custom-designed 3D printed case to house the circuit board. I developed a Python library to accompany the device, which allows the end user to interact with the sensors in Python. The user can connect four NXT sensors to the LinkSense and use them to receive information about the Linkbot's surroundings.
